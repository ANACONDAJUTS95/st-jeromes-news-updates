# Jeromian Voice

**Jeromian Voice** is the official digital news and updates platform for **St. Jerome's Academy**. It serves as a modern scholarly archive and a digital curator for the institution's heritage, research, and campus life.

## 🏛️ Project Vision: The Digital Curator

Moving away from sterile, grid-locked designs, *Jeromian Voice* embraces the storied layouts of high-end editorial broadsheets. It utilizes **Intentional Asymmetry** and **Bold Minimalism** to evoke the feeling of a prestigious academic institution that respects its heritage while embracing the speed of modern information.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com)
- **Typography:** 
  - **Serif:** *Newsreader* (Voice of Authority)
  - **Sans:** *Inter / Roboto* (Modern Utility)
- **Animations:** Custom Tailwind reveal-on-scroll and CSS clip-path transitions.

## ✨ Core Features

- **The Newsroom:** A magazine-style dispatch center for breaking updates and research briefings.
- **The Archives:** A high-contrast library catalog for digital manuscripts and historical preservation.
- **Student Spotlight:** Immersive, long-form interview layouts celebrating the academy's modern scholars.
- **Responsive Mastery:** Fully optimized for all viewports from 375px to ultra-wide displays.
- **Interactive UI:** Smooth page transitions, sticky glassmorphic navigation, and state-driven animations.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live prototype.

## 🎨 Design System

The project's visual logic is governed by:
